# 硬件接线指南

## 📋 材料清单

### 必需组件

1. **Arduino开发板** ×1
  - Arduino UNO
  - Arduino Nano
  - 或任何兼容的开发板
2. **RGB LED灯** ×1
  - 共阳极（Common Anode）或共阴极（Common Cathode）
  - 建议使用透明雾状LED，效果更好
3. **220Ω电阻** ×3
  - 限流保护LED
  - 也可使用330Ω电阻
4. **面包板** ×1
  - 方便原型搭建
5. **杜邦线** ×若干
  - 公对公或公对母
6. **USB数据线** ×1
  - 连接电脑和Arduino

### 可选组件

- **LED扩散罩** - 使光线更柔和
- **外壳** - 3D打印或自制
- **蜂鸣器** - 添加声音提示
- **按钮** - 手动控制开关

## 🔌 接线图

### 方案一：共阳极RGB LED（推荐）

共阳极RGB LED的**长脚为正极**，需要连接到**5V**。

```
Arduino UNO/Nano         共阳极RGB LED
───────────────────────────────────────
 Digital Pin 9 (PWM)  ──── R (红色引脚) ──┐
                                           │
 Digital Pin 10 (PWM) ──── G (绿色引脚) ──┤ 各串联
                                           │  220Ω电阻
 Digital Pin 11 (PWM) ──── B (蓝色引脚) ──┘
                                           │
 5V                  ──── 公共阳极 (+) ──────┘
                                           │
 GND                 ──── 不需要连接       │
```

**电路图：**

```

    │ └───┘   └───┘   │   │

    │ ┌───┐   ┌───┐   │   │ │
```

### 方案二：共阴极RGB LED

共阴极RGB LED的**长脚为负极**，需要连接到**GND**。

```
Arduino UNO/Nano         共阴极RGB LED
───────────────────────────────────────
 Digital Pin 9 (PWM)  ──── R (红色引脚) ──┐
                                           │
 Digital Pin 10 (PWM) ──── G (绿色引脚) ──┤ 各串联
                                           │  220Ω电阻
 Digital Pin 11 (PWM) ──── B (蓝色引脚) ──┘
                                           │
 GND                 ──── 公共阴极 (-) ──────┘
                                           │
 5V                  ──── 不需要连接       │
```

**代码调整：**
对于共阴极LED，需要修改Arduino代码中的`setColor`函数：

```cpp
// 共阴极：直接设置PWM值
void setColor(int red, int green, int blue) {
  analogWrite(PIN_RED, red);
  analogWrite(PIN_GREEN, green);
  analogWrite(PIN_BLUE, blue);
}
```

## 🔍 引脚识别

### RGB LED引脚识别

```
       ┌───┐
      ─┤   │─ 公共端（最长引脚）
       │   │
    R ─┤   │─ G
       │   │
    B ─┤   │─ 空引脚（可能没有）
       └───┘

常见引脚排列（从长脚开始逆时针）：
1. 公共端（最长）
2. 红色（R）
3. 公共端或空引脚
4. 绿色（G）
5. 蓝色（B）
```

**不确定时**：使用万用表测试，或参考LED的数据手册。

### Arduino引脚说明

- **9、10、11引脚**：支持PWM，可调节亮度
- **5V**：电源输出
- **GND**：接地

## 🛠️ 搭建步骤

### 步骤1：准备组件

1. 清点所有组件
2. 确认RGB LED类型（共阳/共阴）
3. 准备电阻和杜邦线

### 步骤2：搭建电路

1. 将Arduino插入面包板
2. 插入RGB LED，注意方向
3. 为每个颜色引脚串联220Ω电阻
4. 连接杜邦线：
  - 红色引脚 → 电阻 → Arduino Pin 9
  - 绿色引脚 → 电阻 → Arduino Pin 10
  - 蓝色引脚 → 电阻 → Arduino Pin 11
  - 公共端 → Arduino 5V（共阳）或 GND（共阴）

### 步骤3：检查连接

1. 检查所有连接是否牢固
2. 确保没有短路
3. 确认电阻值正确

### 步骤4：供电测试

1. 连接USB线到电脑
2. Arduino电源灯应亮起
3. RGB LED可能会微亮（正常现象）

## 🧪 测试电路

### 简单测试（不使用代码）

1. 用导线短暂连接5V到红色引脚（通过电阻）
2. LED应亮红色
3. 同样测试绿色和蓝色

### 代码测试

1. 上传测试代码：
  ```cpp
   void setup() {
     pinMode(9, OUTPUT);
     pinMode(10, OUTPUT);
     pinMode(11, OUTPUT);
   }

   void loop() {
     digitalWrite(9, HIGH); delay(1000); digitalWrite(9, LOW);
     digitalWrite(10, HIGH); delay(1000); digitalWrite(10, LOW);
     digitalWrite(11, HIGH); delay(1000); digitalWrite(11, LOW);
   }
  ```
2. 观察RGB LED是否依次亮起红、绿、蓝

## ⚠️ 安全注意事项

1. **防止短路**
  - 确保导线绝缘良好
  - 避免金属部分接触
2. **正确使用电阻**
  - 必须串联电阻，否则会烧毁LED
  - 电阻值：180Ω-330Ω均可
3. **电源安全**
  - 使用电脑USB供电（5V，500mA）
  - 不要使用外部电源，除非你知道在做什么
4. **静电防护**
  - 触摸金属物体放电后再操作
  - 避免在干燥环境下操作

## 🔧 故障排除

### LED完全不亮

1. 检查电源：Arduino电源灯是否亮？
2. 检查公共端：是否连接到5V（共阳）或GND（共阴）？
3. 检查LED方向：是否接反？
4. 测试LED：直接连接3V电池测试（串联电阻）

### 只有某些颜色亮

1. 检查电阻连接：是否每个颜色都有电阻？
2. 检查引脚连接：是否连接到正确的Arduino引脚？
3. 检查代码：PWM引脚定义是否正确？

### LED亮度异常

1. 太暗：电阻值可能太大，尝试220Ω
2. 太亮/发热：电阻值太小，增加电阻值
3. 颜色不正：检查PWM值，共阳/共阴配置

### Arduino无法识别

1. 检查USB线：是否数据线（有些只供电）
2. 检查端口：设备管理器中查看COM端口
3. 检查驱动：安装Arduino驱动
